// Divide una cadena en dos donde encuentre 'breaker'.
procedure LiteralBreak(string input, string breaker, string out1, string out2);
si input empieza por breaker
input = breaker + out1 + out2
out2 comienza por breaker
si input NO empieza por breaker
input = out1 + out2
out2 comienza por breaker

Si no encuentra breaker devuelve en out1 la cadena vacía y en out2 input.

// Divide una cadena en dos donde encuentre 'breaker'.
procedure LiteralSeparate(string input, string breaker, string out1, string out2);
si input empieza por breaker
input = out1+ breaker + out2

Si no encuentra breaker devuelve en out1 input y en out2 la cadena vacía.
